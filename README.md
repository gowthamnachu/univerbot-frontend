# UniverBot Frontend

Next.js 14 frontend for the UniverBot AI Chatbot Builder platform.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Dark theme with cyan accent (#00E5FF)
- Supabase authentication
- Bot creation and management dashboard
- Document upload for training
- API key management
- Embeddable widget code generation

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
